# 安装部署说明

## 直接访问服务器

由于我们已经把程序部署在服务器上，因此可直接访问123.207.110.18:8081。

## 本地部署

1. 安装node.js：`npm install node`
2. 切换到Color-Bayan目录，执行`npm install`
3. 安装gulp：
   1. `npm install --global gulp`
   2. `npm install --save-dev gulp`
4. 测试gulp：`gulp -v`
5. 执行gulp 命令行输入： `gulp`
6. 双击打开 /dist 目录下的 index.html 即可看到相应的网页（需要联网）。
